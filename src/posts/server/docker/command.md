## 帮助/启动类命令

-   启动 docker：

```shell
systemctl start docker
```

-   停止 docker：

```shell
systemctl stop docker`
```

-   重启 docker：

```shell
systemctl restart docker
```

-   查看 docker 状态：

```shell
systemctl status docker
```

-   开机启动：

```shell
systemctl enable docker
```

-   查看 docker 概要信息：

```shell
docker info
```

-   查看 docker 总体帮助文档：

```shell
docker --help
```

-   查看 docker 命令帮助文档：

```shell
docker 具体命令 --help
```
